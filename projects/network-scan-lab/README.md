# Network Scan Lab

## Goal
Practice discovering devices and services on a network.

## Tools
- Nmap
- Linux terminal

## Setup
- One Linux VM
- One target machine (or same VM)

## Steps
1. Ran basic Nmap scan
2. Identified open ports
3. Detected services

## Findings
- List open ports
- List services found

## What I Learned
- How scanning works
- How attackers and defenders use scans
